Este projeto consiste na modelagem de um banco de dados para um sistema de gerenciamento de ordens de serviço (OS) de uma oficina mecânica.

O objetivo é organizar e controlar todo o processo de manutenção de veículos, desde a entrada do cliente na oficina até a conclusão dos serviços, permitindo o gerenciamento de clientes, veículos, equipes de mecânicos, ordens de serviço, peças utilizadas e mão de obra.

Narrativa do Problema

Clientes levam seus veículos à oficina mecânica para realização de consertos ou revisões periódicas.

Cada veículo é designado a uma equipe de mecânicos responsável por avaliar o problema, identificar os serviços necessários e emitir uma Ordem de Serviço (OS), contendo a previsão de entrega do veículo.

Com base na Ordem de Serviço, é calculado o valor dos serviços utilizando uma tabela de referência de mão de obra. Além disso, o custo das peças utilizadas também compõe o valor total da OS.

Antes do início dos trabalhos, o cliente deve autorizar a execução dos serviços.

A mesma equipe responsável pela avaliação realiza a execução dos serviços até sua conclusão.
